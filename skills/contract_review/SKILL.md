## 📋 技能1：合同类型识别与确认

### 识别方法

当用户上传文档时，通过以下方式判断合同类型：

1. **标题关键词识别**：
   - 劳动合同 → "劳动合同"、"劳务派遣"、"聘用协议"
   - 采购合同 → "采购"、"买卖"、"供货"、"购销"
   - 服务合同 → "服务"、"咨询"、"技术"、"委托"
   - 租赁合同 → "租赁"、"租用"、"房租"、"设备租赁"
   - 金融合同 → "贷款"、"借款"、"授信"、"保理"
   - 建筑工程合同 → "施工"、"建设"、"工程"、"装修"

2. **主体内容识别**：
   - 劳动关系特征：工资、社保、工作内容、管理关系
   - 采购特征：货物/设备名称、数量、价格、交付
   - 服务特征：服务内容、成果交付、服务费
   - 金融特征：利率、本金、还款、担保

3. **条款关键词识别**：
   - "工作时间"、"加班费"、"社保" → 劳动合同
   - "验收"、"质保"、"交货" → 采购合同
   - "服务费"、"成果物"、"知识产权" → 服务合同

### 互动确认机制

若无法明确识别，必须主动与用户确认：

> *"您上传的合同是服务合同（如技术服务、咨询服务协议）还是采购合同（如商品买卖、设备采购协议）？请补充说明合同类型，以便精准匹配合规标准。"*

> *"您的合同是否涉及劳动关系？是否包含工程建设相关内容？是否涉及贷款/金融业务？请协助明确，以便精准匹配审查标准。"*

### 确认后处理

确认类型后，明确标注合同类型，进入对应合规审查流程。

## 📋 技能2：合规标准匹配与条款审查

### 一、法律标准匹配

根据合同类型，匹配对应的法律及专项法规：

| 合同类型 | 通用法律依据 | 专项法规依据 |
|---------|-------------|-------------|
| 通用合同 | 《民法典》合同编、《民法典》总则编 | - |
| 采购/销售合同 | 《民法典》合同编、《产品质量法》 | 《消费者权益保护法》 |
| 金融服务合同 | 《民法典》合同编 | 《商业银行法》、《贷款通则》、《银行业监督管理法》 |
| 建筑工程合同 | 《民法典》合同编 | 《建筑法》、《建设工程质量管理条例》 |
| 劳动合同 | 《劳动合同法》 | 《劳务派遣暂行规定》 |
| 租赁合同 | 《民法典》合同编（租赁合同章） | 《城市房地产管理法》 |
| 技术合同 | 《民法典》合同编 | 《专利法》、《著作权法》 |
| 数据/信息安全 | 《数据安全法》、《个人信息保护法》 | 《网络安全法》、等保2.0 |

### 二、合同条款审查要点

#### 2.1 主体资格审查

| 检查项 | 检查内容 | 法律依据 |
|-------|---------|---------|
| 企业主体资格 | 营业执照是否有效、是否在存续期内 | 《民法典》第58条 |
| 法定代表人权限 | 签字人是否为法定代表人或授权代表 | 《民法典》第61条 |
| 授权委托书 | 是否有合法有效的授权委托书 | 《民法典》第165条 |
| 特殊资质 | 行业特许经营许可（如金融牌照、建筑资质） | 各行业监管规定 |
| 签约能力 | 是否具备民事行为能力 | 《民法典》第21-24条 |

**审查话术**：
> *"经审查，签约方【XXX公司】具备合法企业法人资格，营业执照有效存续。签字人【XXX】为法定代表人/持有有效授权委托书，符合签约资格要求。"*

#### 2.2 权利义务审查

| 检查项 | 检查内容 | 风险等级 |
|-------|---------|---------|
| 权责对等性 | 双方权利义务是否对等 | 高 |
| 无限责任条款 | 是否存在"一方全责"、"一方无限责任" | 高 |
| 排除法定权利 | 是否排除对方法定解除权、抗辩权 | 高 |
| 义务范围 | 义务范围是否超出法律规定 | 中 |
| 违约金比例 | 违约金是否过高（>合同金额30%） | 中 |

**高风险条款识别**：
- ❌ "一方对另一方损失承担全部赔偿责任"
- ❌ "一方不得提前解除合同"
- ❌ "一方放弃抗辩权"
- ❌ "违约金为合同总金额的50%"

#### 2.3 核心条款完整性

| 必备条款 | 说明 | 缺失风险 |
|---------|-----|---------|
| 标的描述 | 合同标的物/服务内容明确 | 高 |
| 数量/质量 | 数量明确，质量标准具体 | 高 |
| 价款/支付 | 金额明确，支付方式/节点清晰 | 高 |
| 履行期限 | 交货/服务/还款时间明确 | 中 |
| 违约责任 | 违约情形及责任承担具体 | 高 |
| 争议解决 | 管辖法院/仲裁机构明确 | 中 |

#### 2.4 特殊风险条款审查

##### （1）格式条款审查

| 检查项 | 检查要求 | 法律依据 |
|-------|---------|---------|
| 提示义务 | 是否以合理方式提示对方注意免除/限制其责任的条款 | 《民法典》第496条 |
| 说明义务 | 是否对格式条款进行说明 | 《民法典》第496条 |
| 无效格式条款 | 是否存在排除对方主要权利、减轻/免除己方责任的条款 | 《民法典》第497条 |

**话术**：
> *"【风险提示】合同第X条属于格式条款，但未以加粗/下划线等方式提示对方注意。根据《民法典》第496条，建议补充提示义务条款，否则该条款可能被认定为无效。"*

##### （2）保密条款审查

| 检查项 | 检查内容 |
|-------|---------|
| 保密范围 | 是否明确界定商业秘密/保密信息范围 |
| 保密期限 | 期限是否合理（通常≥合同履行完毕后2年） |
| 违约责任 | 泄密违约责任是否具体可执行 |
| 个人信息 | 如涉及个人信息，是否符合《个人信息保护法》要求 |

##### （3）知识产权条款审查

| 检查项 | 检查内容 |
|-------|---------|
| 权属约定 | 成果物知识产权归属是否明确 |
| 授权范围 | 授权使用范围、期限、地域是否清晰 |
| 侵权责任 | 知识产权侵权责任承担是否明确 |
| 第三方权利 | 是否涉及第三方知识产权（开源组件、专利等） |

##### （4）违约责任条款审查

| 检查项 | 检查内容 |
|-------|---------|
| 违约金比例 | 违约金是否过高（建议≤30%） |
| 损失赔偿范围 | 赔偿范围是否明确，是否包括间接损失 |
| 免责条款 | 免责条款是否合理，是否排除故意/重大过失 |
| 责任上限 | 是否有责任上限约定，是否合理 |

### 三、签约方资质核验

**重要提醒**：若合同中出现除"重庆农村商业银行"（或用户所在银行）以外的企业/机构名称，需进行资质核验：

| 核验项目 | 检查内容 |
|---------|---------|
| 存续状态 | 企业是否有效存续（未被吊销/注销） |
| 签约资质 | 是否具备签约所需的特殊资质/许可证 |
| 法律纠纷 | 是否存在重大诉讼/被执行记录 |
| 行政处罚 | 是否存在近期行政处罚记录 |

**核验方式**：
- 调用企业信息查询工具（如天眼查、企查查等）
- 核实营业执照、许可证原件
- 查询裁判文书网、中国执行信息公开网

## 五、改进建议 📝

### 建议1：调整第5条解除权条款
- **当前条款**："乙方不得提前终止合同"
- **问题**：排除法定解除权，违反《民法典》第563条
- **修改建议**：
  > "乙方有权提前30日书面通知甲方解除合同。乙方提前解除合同的，应当按合同总金额的X%向甲方支付违约金。"
- **法律依据**：《民法典》第563条、第565条

### 建议2：调整第8条违约金比例
- **当前条款**："违约金为合同总金额的50%"
- **问题**：违约金比例过高，可能被法院调减
- **修改建议**：
  > "违约金为合同总金额的15%，最高不超过合同总金额的30%。"
- **法律依据**：《民法典》第585条

## 审查限制与注意事项

### ⚠️ 重要限制

1. **仅受理合同类文档**：仅接受 docx、pdf、txt 等格式的合同文档
2. **必须明确合同类型**：若无法识别，需主动与用户确认
3. **必须引用法律依据**：所有审查结论必须引用具体法规条款
4. **不确定条款提示**：对存在法律适用不确定的条款，需提示用户咨询专业律师

### ⚠️ 审查边界

- ❌ 不审查非合同类文档（个人信件、memo等）
- ❌ 不提供法律咨询（仅做合规性审查）
- ❌ 不替代律师意见（重大条款建议咨询律师）

### ⚠️ 责任声明

> *"本审查报告基于合同文本内容进行合规性分析，仅供内部参考使用，不构成法律意见。合同中涉及的具体法律问题，特别是重大风险条款，建议用户在签署前咨询专业律师，以确保合法权益得到充分保障。"*

## 常用法律依据速查

### 《民法典》核心条款

| 条款 | 内容 | 适用场景 |
|-----|------|---------|
| 第143条 | 民事法律行为有效要件 | 合同效力判断 |
| 第496条 | 格式条款提示说明义务 | 格式条款审查 |
| 第497条 | 格式条款无效情形 | 格式条款效力 |
| 第563条 | 法定解除权 | 合同解除权审查 |
| 第565条 | 解除合同程序 | 解除权行使 |
| 第585条 | 违约金上限 | 违约金调整 |
