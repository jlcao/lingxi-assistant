# 工具沙盒集成示例

## 概述

本示例展示如何将现有的 `ToolBase` 系列工具（FileTool、CommandTool、ReadSkillTool）与新创建的分级沙盒系统集成。

---

## 架构关系

```
现有工具架构          新沙盒系统
─────────────────────────────────────
ToolBase              SandboxManager
  ├─ FileTool    ←→    ├─ L1Sandbox (线程)
  ├─ CommandTool ←→    └─ L2Sandbox (进程+venv)
  └─ ReadSkillTool
           ↓
    ToolSandboxAdapter (适配器)
```

---

## 集成方案

### 方案 1：使用 ToolSandboxAdapter（推荐）

```python
from lingxi.skills import ToolSandboxAdapter, TrustLevel
from lingxi.core.utils.Tool import Tool

# 1. 初始化工具管理器
tool_manager = Tool(skill_system=None)

# 2. 初始化沙盒适配器
adapter = ToolSandboxAdapter()

# 3. 设置工具的信任等级（file 和 command 工具默认 L2）
adapter.set_tool_trust_level("file", TrustLevel.L2)
adapter.set_tool_trust_level("command", TrustLevel.L2)
adapter.set_tool_trust_level("read_skill", TrustLevel.L1)

# 4. 在沙盒中执行工具
file_tool = tool_manager.tools["file"]

# 执行文件读取操作
params = {
    "operation_type": "read",
    "file_path": "test.txt"
}

result = adapter.execute_tool_in_sandbox(
    file_tool.execute,
    params,
    tool_name="file"
)

print(result.success)    # True/False
print(result.data)       # 工具返回内容
print(result.message)    # 提示信息
```

### 方案 2：使用 adapt_tool_manager 快速适配

```python
from lingxi.skills import adapt_tool_manager, TrustLevel
from lingxi.core.utils.Tool import Tool

# 1. 初始化工具管理器
tool_manager = Tool(skill_system=None)

# 2. 快速适配（默认 file/command 为 L2，其他为 L1）
adapter = adapt_tool_manager(tool_manager)

# 3. 执行工具
file_tool = tool_manager.tools["file"]
params = {"operation_type": "read", "file_path": "test.txt"}

result = adapter.execute_tool_in_sandbox(
    file_tool.execute,
    params,
    tool_name="file"
)
```

### 方案 3：创建沙盒化的工具函数

```python
from lingxi.skills import create_sandboxed_tool_execute
from lingxi.core.utils.FileTool import FileTool

# 1. 创建工具实例
file_tool = FileTool()

# 2. 创建沙盒化的执行函数
sandboxed_execute = create_sandboxed_tool_execute(file_tool)

# 3. 直接调用（自动在沙盒中执行）
params = {"operation_type": "read", "file_path": "test.txt"}
result = sandboxed_execute(params)
```

## 信任等级配置

| 工具 | 默认信任等级 | 说明 |
|------|------------|------|
| `file` | L2 | 文件操作，高风险 |
| `command` | L2 | 命令执行，高风险 |
| `read_skill` | L1 | 读取技能说明，低风险 |

### 自定义信任等级

```python
from lingxi.skills import TrustLevel

# 设置特定工具的信任等级
adapter.set_tool_trust_level("my_custom_tool", TrustLevel.L1)

# 执行时临时指定信任等级
result = adapter.execute_tool_in_sandbox(
    tool_execute,
    params,
    tool_name="file",
    trust_level=TrustLevel.L1  # 临时降级
)
```

## 完整示例

```python
import sys
import os
sys.path.insert(0, os.path.dirname(os.path.abspath(__file__)))

from lingxi.skills import (
    ToolSandboxAdapter,
    adapt_tool_manager,
    TrustLevel
)
from lingxi.core.utils.Tool import Tool
from lingxi.core.utils.FileTool import FileTool


def example_1_basic_usage():
    """示例 1: 基础使用"""
    print("=== 示例 1: 基础使用 ===")
    
    # 初始化工具管理器
    tool_manager = Tool(skill_system=None)
    
    # 初始化适配器
    adapter = ToolSandboxAdapter()
    
    # 获取 FileTool
    file_tool = tool_manager.tools["file"]
    
    # 创建测试文件
    test_file = "./sandbox_test.txt"
    with open(test_file, "w", encoding="utf-8") as f:
        f.write("第一行内容\n第二行内容")
    
    # 在沙盒中读取文件
    read_params = {
        "operation_type": "read",
        "file_path": test_file
    }
    
    result = adapter.execute_tool_in_sandbox(
        file_tool.execute,
        read_params,
        tool_name="file"
    )
    
    print(f"成功: {result.success}")
    print(f"数据: {result.data}")
    
    # 清理
    os.remove(test_file)
    adapter.shutdown()


def example_2_adapt_tool_manager():
    """示例 2: 使用 adapt_tool_manager"""
    print("\n=== 示例 2: 使用 adapt_tool_manager ===")
    
    tool_manager = Tool(skill_system=None)
    adapter = adapt_tool_manager(tool_manager)
    
    # 查看各工具的信任等级
    for tool_name in tool_manager.tools:
        trust_level = adapter.get_tool_trust_level(tool_name)
        print(f"工具 {tool_name}: 信任等级 {trust_level}")
    
    adapter.shutdown()


def example_3_custom_trust_level():
    """示例 3: 自定义信任等级"""
    print("\n=== 示例 3: 自定义信任等级 ===")
    
    adapter = ToolSandboxAdapter(default_trust_level=TrustLevel.L2)
    
    # 设置 FileTool 为 L1（临时降级）
    adapter.set_tool_trust_level("file", TrustLevel.L1)
    
    file_tool = FileTool()
    test_file = "./sandbox_test2.txt"
    
    with open(test_file, "w", encoding="utf-8") as f:
        f.write("测试内容")
    
    params = {
        "operation_type": "read",
        "file_path": test_file
    }
    
    result = adapter.execute_tool_in_sandbox(
        file_tool.execute,
        params,
        tool_name="file"
    )
    
    print(f"执行结果: {result.success}")
    
    os.remove(test_file)
    adapter.shutdown()


if __name__ == "__main__":
    example_1_basic_usage()
    example_2_adapt_tool_manager()
    example_3_custom_trust_level()

```

## API 参考

### ToolSandboxAdapter

| 方法 | 说明 |
|------|------|
| `__init__(sandbox_manager, default_trust_level)` | 初始化适配器 |
| `set_tool_trust_level(tool_name, trust_level)` | 设置工具信任等级 |
| `get_tool_trust_level(tool_name)` | 获取工具信任等级 |
| `wrap_tool_execute(tool_execute, tool_name, ...)` | 包装工具执行函数 |
| `execute_tool_in_sandbox(tool_execute, params, ...)` | 在沙盒中执行工具 |
| `shutdown(wait)` | 关闭沙盒 |

### 辅助函数

| 函数 | 说明 |
|------|------|
| `adapt_tool_manager(tool_manager, ...)` | 快速适配 Tool 管理器 |
| `create_sandboxed_tool_execute(tool, adapter)` | 创建沙盒化的工具函数 |
