# L2 沙箱技能规范

## 概述

本文档定义了 L2 沙箱（进程级隔离）技能的 `main.py` 必须遵循的规范。

## 核心要求

### 1. 必须有 `execute(parameters: Dict[str, Any]) -> str` 函数

```python
def execute(parameters: Dict[str, Any]) -> str:
    """执行技能
    Args:
        parameters: 参数字典
    Returns:
        结果字符串
    """
    # 实现逻辑
    return "结果"
```

### 2. 必须有 `__main__` 入口点

```python
if __name__ == "__main__":
    import argparse
    import json
    
    parser = argparse.ArgumentParser()
    parser.add_argument("--params", type=str, required=True, help="参数文件路径")
    args = parser.parse_args()
    
    with open(args.params, "r", encoding="utf-8") as f:
        parameters = json.load(f)
    
    result = execute(parameters)
    print(result)
```

### 3. 完整示例

```python
#!/usr/bin/env python3
"""技能描述"""

import logging
from typing import Dict, Any

def execute(parameters: Dict[str, Any]) -> str:
    """执行技能"""
    logger = logging.getLogger(__name__)
    logger.info(f"执行技能，参数: {parameters}")
    
    # 实现逻辑
    return "执行结果"

if __name__ == "__main__":
    import argparse
    import json
    
    logging.basicConfig(level=logging.INFO)
    
    parser = argparse.ArgumentParser()
    parser.add_argument("--params", type=str, required=True, help="参数文件路径")
    args = parser.parse_args()
    
    with open(args.params, "r", encoding="utf-8") as f:
        parameters = json.load(f)
    
    result = execute(parameters)
    print(result)
```
