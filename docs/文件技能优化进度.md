## 📝 使用示例

### read_file 新用法

```python
# 流式读取大文件
read_file(
    file_path="/var/log/large.log",
    stream=True,
    max_lines=5000
)

# 自动检测编码读取 GBK 文件
read_file(
    file_path="/path/to/gbk_file.txt",
    detect_encoding=True
)

# 指定编码读取
read_file(
    file_path="/path/to/file.txt",
    encoding="gbk",
    detect_encoding=False
)
```

### create_file 新用法

```python
# 追加内容到文件
create_file(
    file_path="/path/to/log.txt",
    content="\nNew log entry",
    mode="append"
)
```

### apply_patch 用法

```python
# 预览补丁
apply_patch(
    file_path="/path/to/file.py",
    patch_text="...",
    dry_run=True
)

# 应用补丁并创建备份
apply_patch(
    file_path="/path/to/file.py",
    patch_text="...",
    backup=True
)
```

### batch_read 用法

```python
# 批量读取 Python 文件
batch_read(
    pattern="*.py",
    directory="/path/to/project",
    max_files=20
)

# 读取指定文件列表
batch_read(
    file_paths=[
        "/path/to/file1.txt",
        "/path/to/file2.txt"
    ]
)
```
